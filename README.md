# wp-sitebuilderone

All-in-One plugin for local business websites

## Requirements

1. LiveCanvas plugin
2. Picostrap child theme
3. ACF Pro (Options)
3. Git-updater https://github.com/afragen/git-updater


## Setup

1. Install Git-updater
2. Point to https://github.com/sitebuilderone/wp-sitebuilderone
